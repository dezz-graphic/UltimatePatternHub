============================================================

ขอบคุณที่สนับสนุนโปรแกรม Ultimate Pattern Hub ครับ! 
โปรแกรมนี้เป็นระบบลิขสิทธิ์แท้แบบใช้งานตลอดชีพ (1 เครื่อง ต่อ 1 ลิขสิทธิ์) 
กรุณาทำตามขั้นตอนด้านล่างเพื่อปลดล็อกการใช้งาน:

สเตปที่ 1: การติดตั้ง (Installation)
------------------------------------------------------------
1. ปิดโปรแกรม Adobe Illustrator ก่อนเริ่มการติดตั้ง
2. ดับเบิลคลิกไฟล์ "UltimatePatternHub_Installer.exe"
3. ทำตามขั้นตอนบนหน้าจอ (Next > Install) จนเสร็จสมบูรณ์
   *หาก Windows ถามสิทธิ์การติดตั้ง (UAC) ให้กด "Yes" เพื่ออนุญาต*

สเตปที่ 2: การเปิดโปรแกรมครั้งแรก (Activation)
------------------------------------------------------------
1. เปิดโปรแกรม Adobe Illustrator
2. ไปที่เมนูด้านบน: Window > Extensions > Ultimate Pattern Hub
3. หน้าจอจะปรากฏหน้าต่าง Lock Screen พร้อมระบุรหัส "MACHINE ID"

สเตปที่ 3: การขอรหัสปลดล็อก (Request License)
------------------------------------------------------------
1. ให้คุณทำการ "คัดลอก (Copy) รหัส MACHINE ID" ที่ปรากฏบนหน้าจอ
2. ส่งรหัสดังกล่าวกลับมาให้แอดมินทางแชท หรือกดปุ่ม 
   "Contact to Get License Key" เพื่อทักหาแอดมินโดยตรง
3. รอรับรหัส "LICENSE KEY" จากแอดมิน

สเตปที่ 4: การปลดล็อกใช้งาน (Unlock)
------------------------------------------------------------
1. นำ LICENSE KEY ที่ได้รับจากแอดมิน มากรอกในช่อง Enter License Key
2. กดปุ่ม "UNLOCK"
3. ระบบจะปลดล็อกหน้าจอ และคุณสามารถเริ่มใช้งานได้ทันที!

------------------------------------------------------------
⚠️ หมายเหตุสำคัญ:
- รหัส License Key จะผูกติดกับคอมพิวเตอร์เครื่องนี้เท่านั้น
- หากมีการเปลี่ยนอุปกรณ์ หรือต้องการย้ายเครื่อง กรุณาติดต่อแอดมิน
- ห้ามลบโฟลเดอร์โปรแกรมใน Common Files เพื่อป้องกันการตั้งค่าหาย
